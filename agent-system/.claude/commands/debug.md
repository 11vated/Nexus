# /debug Command

Use when encountering errors or test failures.

## Process
1. Read error message carefully
2. Find relevant code
3. Understand the root cause
4. Fix with minimal changes
5. Verify fix

## Common Fixes
- ImportError → check package installed
- TypeError → check variable types
- AttributeError → check object has attribute
- KeyError → check dictionary has key
