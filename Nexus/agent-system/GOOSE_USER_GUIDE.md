## Troubleshooting

### If Goose seems robotic:
- Restart the app
- The system prompt has been optimized for conversational behavior

### If model won't load:
- Make sure Ollama is running: curl http://localhost:11434/api/version
- Check model is installed: ollama list

### If extensions don't work:
- Some require API keys (Notion, Slack, etc.)
- Built-in extensions always work
