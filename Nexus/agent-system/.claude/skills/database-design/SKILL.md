# Database Design Skill

## Principles
- Normalize to 3NF
- Use appropriate data types
- Add indexes for queries

## Types
- INTEGER for IDs
- VARCHAR(n) for strings
- TEXT for large text
- BOOLEAN for flags
- TIMESTAMP for dates

## Tools
- SQLAlchemy (Python)
- SQL migrations
